# 차지차지 (ChajiChaji) — EV 충전소 맞춤 추천 · Backend

Spring Boot 기반의 REST API 서버입니다.  
**한국환경공단**(충전소 위치/운영) 및 **TMAP**(지도, 경로) 데이터를 활용하여  
**사용자 차량 전비, 가격, 경로** 정보를 반영한 충전소 추천을 제공합니다.  
캐시 적용으로 외부 API 호출 응답 속도를 최적화했습니다.
